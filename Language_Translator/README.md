# 🌐 Language Translator

## Overview
This project is a Python-based Language Translator developed as part of the CodeAlpha Internship. It translates text from one language to another using a translation library.

## Features
- Translate text between multiple languages
- Simple and easy-to-use interface
- Fast translation

## Requirements
- Python 3.x
- googletrans (or the library you used)

## How to Run
1. Install the required library.
2. Run:
   python translator.py
