# codealpha_tasks-# 🎵 Music Generation Using Python

## Overview
This project is a Python-based Music Generation application developed as part of the CodeAlpha Internship. It generates MIDI music using a dataset of MIDI files.

## Features
- Generates MIDI music
- Uses a MIDI dataset
- Simple Python implementation
- Produces a generated_music.mid output file

## Technologies Used
- Python
- MIDI Files

## Project Structure
- music.py - Main Python program
- dataset/ - Input MIDI files
- generated_music.mid - Generated music output

## How to Run
1. Clone this repository.
2. Install the required Python libraries.
3. Run `music.py`.
4. The generated music will be saved as `generated_music.mid`.
